HTML


1. Text content
2. Markup (style)
3. References to other documents
4. Links


Markup = Tags < name> content < /name>

Opening, content, closing = Element

< b> = Bold
< em> = Italics
< a> = Anchor (links) My Codepen

< img> = images (no end tag required = void tag)

< br> = text on multiple lines (can enter multiple)
< p> = paragraph
< br> = inline (text only) (Examples: b, em, img, span, br, strong)
< p>= block (invisible box) (div, p, form)

Contain content < span> = inline
< span class=“foo(CSS)”>

< div> = block
< div class=“bar”>
